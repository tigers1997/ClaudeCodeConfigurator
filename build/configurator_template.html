Claude Code project configurator

Fill in the intake form (every field has a default), pick modules, download a setup script. Run it from the root of a target project to scaffold a fully personalized CLAUDE.md, .claude/, and optionally .mcp.json.

1. Intake form

Everything you enter gets substituted into CLAUDE.md via {{placeholder}} keys. Click any section to expand.

2. Modules to install

Each module is a self-contained bundle of files + settings patches. Click "show files" on any card to preview.
